== Description ==

This plugin shows the post count of authors in dashboard.
Currently the list is limited to top ten by countwise. To increase the number of user's listed u can change the file usercount.php.

Line 11-->    . "Limit 0,10";   

Change the value 10 to any number of your choice.

== Installation ==

1. Upload the folder 'User-post-count' to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
